**Key styling per user decision:**
- `font-family: monospace` -- code-like appearance
- `font-size: 0.8rem` -- compact
- `padding: 4px 8px` -- tight padding
- `background: #f1f3f5` -- light gray
- `border-left: 3px solid #e06c75` -- warm red left border matching the vulnerable-line callout
- `border-radius: 3px` -- subtle rounding
- `margin-top: 4px` -- small gap from the file path line
- `overflow: hidden; text-overflow: ellipsis; white-space: nowrap` -- truncate long lines
- `color: #333` -- readable dark text
